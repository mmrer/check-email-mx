Установка:
- python3 -m venv venv
- source venv/bin/activate
- pip3 install -r requirements.txt

Запуск:
- python3 main.py

*примечание: в корне рабочей директории должен быть файл "email.txt" с строками email которые нужно проверить, его скрипт считает построчно и по окончанию будет выведен список с статусами каждого адреса*
